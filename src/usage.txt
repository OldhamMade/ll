ll - a better ls

Usage:
  ll [-a | -A] [-h] [-V] [-S | -t] [-r] [<path>]
  ll --help
  ll --version

Options:
  -a --all           list entries starting with .
  -A --almost-all    list all except . and ..
  -h --human         show filesizes in human-readable format
  -S --size          sort by size (largest first)
  -t --mtime         sort by modified time (most recently modified first)
  -r --reverse       reverse sort order
  -V --no-vcs        do not get VCS status (faster)
  -v --version       Show version
  --help             Show this screen
